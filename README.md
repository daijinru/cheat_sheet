## Operation

项目根目录创建虚拟环境文件夹

```
$ virtualenv -p python3 venv
```

启动虚拟环境

```
$ . venv/bin/activate
```

安装 Flask

```
(venv) $ pip3 install flask
```

安装 Pymongo

```
(venv) $ pip3 install pymongo
```

启动项目

```
(venv) $ python3 run.py
```
